## 核心论文定位

本文不应写成“换一个深度学习网络预测 VB 值”。更稳妥、更有论文意义的主线是：

```text
原始多传感器信号
-> 弱监督工况分割
-> 提取 stable_cutting 稳定切削段
-> 少样本刀具磨损 VB 回归预测
```

规则伪标签算法不是最终贡献，它只是弱监督标签来源。分割模型的意义在于学习一个可部署的前端工况识别器，用于实时识别有效稳定切削区，并为后续 VB 回归提供更干净、更稳定的输入。

论文最终要证明的不是：

```text
模型能复现规则标签
```

而是：

```text
使用模型分割得到的 stable_cutting 段后，VB 预测误差低于完整波形输入。
```

## 数据集边界与论文口径

PHM2010 每个 CSV 基本是一次加工过程，不是完整机床运行周期，因此它天然不适合强行做完整的：

```text
non_cutting / entry_transition / stable_cutting / exit_transition / retract
```

当前 PHM2010 实验阶段采用二分类：

```text
0: transition
1: stable_cutting
```

`non_cutting` 和 `retract` 不在 PHM2010 主实验中硬造标签。论文中应明确说明：

```text
PHM2010 缺少完整空走、停机、退刀等连续加工周期片段，因此本文首先在 PHM2010 上验证切削过程内部状态分割，即 transition 与 stable_cutting。完整 non_cutting/retract 状态识别作为后续自采连续周期数据的扩展验证。
```

后期如果要把论文叙事扩展成完整加工周期状态识别，需要补充小规模自采数据：

```text
主轴空转 / 快速进给 / 进刀 / 稳定切削 / 出刀 / 退刀 / 停机
```

## 拟解决问题

1. 完整加工波形包含进刀/出刀过渡段，直接用于 VB 预测会混入非稳态信息。
2. 现有少样本刀具磨损预测方法多关注回归网络结构，较少显式处理加工过程内状态差异。
3. PHM2010 缺少逐点工况标签，需要用工艺先验构造弱监督伪标签。
4. 仅报告分割 mIoU 不足以证明工程价值，需要验证分割后的稳定切削段能否提升 VB 预测。

## 预期创新点

1. 提出“弱监督工况分割 + 稳定切削段提取 + 少样本 VB 回归”的两阶段刀具磨损预测框架。
2. 构建基于多传感器活动强度和加工先验的伪标签策略，解决 PHM2010 缺少逐点工况标签的问题。
3. 使用 1D 时序语义分割模型识别 `transition / stable_cutting`，并输出可用于实时监测的逐点状态。
4. 通过稳定切削段筛选降低非稳态过渡段对 VB 回归的干扰。
5. 在 leave-one-tool-out 六折交叉验证和少样本比例实验中验证方法泛化能力。

## 实验安排

### 阶段一：数据理解与标签构建

目标：确认 PHM2010 数据结构、信号长度、VB 标签对应关系，并构建可复用工况标签缓存。

任务：

1. 统计 `c1` 到 `c6` 每个 CSV 的采样点数、通道范围和 VB 标签。
2. 绘制典型完整加工波形，观察过渡段和稳定切削段。
3. 使用伪标签规则生成 `transition / stable_cutting` 二分类标签。
4. 人工检查典型样本，尤其是开头、结尾和低能量切削段。
5. 生成标签缓存，训练时不再重复标注。

输出：

```text
phm2010_segmentation/label_cache/
phm2010_segmentation/outputs/random_label_samples/
phm2010_segmentation/outputs/non_cutting_samples/
```

当前状态：基本完成，但每次修改伪标签规则后必须重新构建缓存。

### 阶段二：工况分割模型

目标：训练前端分割器，识别完整波形中的 `transition` 与 `stable_cutting`。

候选模型：

```text
DeepLabV3-1D
U-Net-1D
TCN-Seg
BiLSTM-Seg
```

评价指标：

```text
Point Accuracy
Mean IoU
Sample Mean IoU
Macro F1
Per-class Precision / Recall / IoU / F1
```

必须做的输出：

1. 六折平均指标表。
2. 每折混淆矩阵。
3. 每折 3 到 5 张完整波形预测可视化图。
4. 伪标签与模型预测对比图。

注意：当前 boundary-window 指标已经可用，但还需要完整波形预测可视化，防止模型只在边界窗口表现好。

### 阶段三：稳定切削段提取

目标：为后续 VB 回归构造三类输入数据。

三种输入：

```text
A. 完整波形
B. 规则伪标签 stable_cutting 段
C. 模型预测 stable_cutting 段
```

任务：

1. 加载每个 CSV 的完整多通道信号。
2. 根据规则标签或模型预测标签提取 `stable_cutting` 区间。
3. 对不同长度片段做统一处理，例如中心裁剪、滑窗采样或插值到固定长度。
4. 记录每个样本的 VB 标签、刀具编号、cut 编号、输入来源。

输出：

```text
完整波形回归数据集
规则 stable_cutting 回归数据集
模型 stable_cutting 回归数据集
segment length/statistics 统计表
分割前后信号片段对比图
```

这是下一阶段最高优先级。

### 阶段四：VB 回归实验

候选回归模型：

评价指标：

```text

```

判断标准：

```text

```

### 阶段五：少样本实验

目标：突出“少样本刀具磨损预测”的价值。

建议训练比例：

```text
10% training cuts
20% training cuts
40% training cuts
100% training cuts
```

每个比例下至少比较：

```text
完整波形
模型 stable_cutting
```

希望证明：

```text
训练样本越少，稳定切削段筛选越能减少噪声干扰，对 VB 预测越有帮助。
```

### 阶段六：消融实验

必须包含：

1. 不使用工况分割 vs 使用模型分割。
2. 完整波形 vs 规则 stable_cutting vs 模型 stable_cutting。
3. 不同分割模型对 VB 预测的影响。
4. 不同回归模型对结果的影响。
5. 不同少样本比例下的结果。

优先级最高的消融：


## 论文写作结构

建议按照“问题动机 -> 弱监督分割 -> 稳定切削段提取 -> VB 预测提升”的逻辑组织全文。

### 1. Introduction

重点写清楚为什么完整波形直接预测 VB 不够可靠：

1. 实际加工信号包含进刀、出刀和稳定切削等不同状态。
2. 过渡段信号具有非稳态特征，可能干扰磨损回归模型。
3. PHM2010 等公开数据集缺少逐点工况标签，导致已有方法多直接使用完整波形。
4. 本文提出弱监督工况分割辅助的少样本刀具磨损预测框架。

建议突出贡献：

```text
1. 提出工况分割引导的少样本刀具磨损预测框架。
2. 基于多传感器信号和加工先验构建弱监督伪标签。
3. 使用 1D 时序分割模型提取 stable_cutting 区域。
4. 通过 VB 回归对比验证稳定切削段提取的有效性。
```

### 2. Related Work

建议分四类写：

```text
Tool wear prediction
Process-state recognition in machining
Weakly supervised / pseudo-label learning
Time-series semantic segmentation
```

写法重点：

1. 刀具磨损预测已有很多深度回归模型，但多数直接使用完整波形。
2. 加工状态识别通常独立于磨损预测，较少与 VB 回归形成统一框架。
3. 弱监督伪标签可用于解决逐点状态标签缺失问题。
4. 本文的差异是把工况分割和少样本 VB 预测连接起来。

### 3. Method

建议拆成四个小节：

```text
3.1 Problem formulation
3.2 Weakly supervised process-state pseudo labeling
3.3 1D temporal segmentation model
3.4 Stable-cutting-guided VB regression framework
```

本章要讲清楚：

1. 输入是七通道传感器信号：力、振动、AE。
2. PHM2010 阶段只定义 `transition / stable_cutting` 二分类。
3. 伪标签是弱监督来源，不是最终模型。
4. 分割模型输出逐点状态概率。
5. 只把 `stable_cutting` 区域送入后端 VB 回归模型。

### 4. Experiments

建议实验顺序：

```text
4.1 Dataset and experimental setup
4.2 Pseudo-label generation and visual inspection
4.3 Process-state segmentation results
4.4 Stable-cutting segment extraction
4.5 Tool wear prediction results
4.6 Few-shot experiments
4.7 Ablation study
```

其中最核心的是 `4.5 Tool wear prediction results`，必须比较：

```text
完整波形 -> VB 回归
规则 stable_cutting -> VB 回归
模型 stable_cutting -> VB 回归
```

### 5. Discussion

重点讨论三个问题：

1. 为什么 PHM2010 不强行做完整 `non_cutting/retract` 分类。
2. 为什么规则伪标签不会削弱模型意义，因为最终评价看 VB 预测提升。
3. 当前方法的局限：伪标签依赖工艺先验，完整加工周期状态还需要自采数据验证。

可写成：

```text
Although the pseudo labels are generated by process priors, the segmentation model is not evaluated only by label reproduction. Its practical value is further validated through downstream VB prediction, where stable-cutting extraction reduces non-stationary interference.
```

### 6. Conclusion

结论要对应贡献，不要只重复指标：

1. 本文提出了弱监督工况分割引导的刀具磨损预测框架。
2. PHM2010 上实现了 `transition / stable_cutting` 的有效分割。
3. 稳定切削段提取为少样本 VB 回归提供了更可靠输入。
4. 后续工作将扩展到自采连续加工周期数据，识别 `non_cutting` 和 `retract` 等完整状态。
