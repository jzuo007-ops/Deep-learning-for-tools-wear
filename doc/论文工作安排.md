## 核心论文定位

不要把文章写成“换一个深度学习网络预测 VB 值”。更有意义的方向是：

```text
原始多传感器信号
-> 工况分割：non_cutting / transition / stable_cutting
-> 提取稳定切削段
-> 少样本刀具磨损 VB 回归预测
```

DeepLabV3 或其他语义分割网络作为前端工况识别器，CNN-BiLSTM-Attention、Stacked-BiLSTM 或 PRes-MHSA-SBiLSTM 作为后端 VB 回归模型。论文贡献应强调：真实加工信号并不总是稳定切削状态，先分割有效加工段可以提高磨损预测的可靠性。

## 拟解决问题

1. PHM2010 每次加工波形包含过渡段、稳定切削段，直接用完整波形预测 VB 会引入无效或低质量信息。
2. 现有少样本刀具磨损预测方法多关注回归网络结构，较少考虑实时加工过程中的工况变化。
3. 当前数据缺少逐点工况标签，需要设计可解释的伪标签或少量人工标注策略。

## 预期创新点

1. 提出一个“工况分割 + 磨损回归”的两阶段刀具磨损监测框架。
2. 设计基于多传感器活动强度的三分类伪标签策略，用于标注 `non_cutting`、`transition`、`stable_cutting`。
3. 使用语义分割网络识别稳定切削区，只将稳定切削段送入 VB 回归模型。
4. 对比完整波形预测与稳定切削段预测，证明分割预处理对少样本磨损预测有帮助。
5. 做跨刀具交叉验证，验证方法在不同刀具数据上的泛化能力。

## 实验安排

### 阶段一：数据理解与标签构建

目标：确认 PHM2010 数据结构、信号长度、VB 标签对应关系，并构建可复用工况标签缓存。

任务：

1. 统计 `c1` 到 `c6` 每个 CSV 的采样点数、通道范围和 VB 标签。
2. 绘制典型完整加工波形，观察过渡段和稳定切削段。
3. 改进三分类伪标签规则，避免把连续切削段误标为 `non_cutting`。
4. 生成一次性标签缓存，训练时不再重复标注。

输出：

```text
phm2010_segmentation/label_cache/
phm2010_segmentation/outputs/pseudo_label_example.png
```

### 阶段二：工况分割模型

目标：训练一个前端分割器，识别完整波形中的工况区域。

候选模型：

```text
DeepLabV3-1D
U-Net-1D
TCN-Seg
BiLSTM-Seg
```

评价指标：

```text
Point Accuracy
Mean IoU
Macro F1
Per-class Precision / Recall / F1
```

必须做的对比：

```text
Rule-based pseudo label only
DeepLabV3-1D
U-Net-1D
TCN-Seg
```

### 阶段三：稳定切削段 VB 回归

目标：只使用 `stable_cutting` 区域特征预测 VB 值。

候选回归模型：

```text
Stacked-BiLSTM
CNN-BiLSTM-Attention
PRes-MHSA-SBiLSTM
```

评价指标：

```text
MAE
RMSE
R2
MAPE
```

对比实验：

```text
完整波形 -> VB 回归
规则稳定切削段 -> VB 回归
DeepLab 分割稳定切削段 -> VB 回归
```

### 阶段四：消融实验

必须包含：

1. 是否使用工况分割。
2. 是否去除 transition 段。
3. 不同伪标签参数对结果的影响。
4. 不同回归主干网络对结果的影响。
5. 不同训练样本数量下的少样本性能。

建议少样本设置：

```text
10% training cuts
20% training cuts
40% training cuts
Full training cuts
```

### 阶段五：交叉验证

使用六组刀具数据做 leave-one-tool-out 交叉验证：

```text
fold c1: test c1, val c2, train c3/c4/c5/c6
fold c2: test c2, val c3, train c1/c4/c5/c6
fold c3: test c3, val c4, train c1/c2/c5/c6
fold c4: test c4, val c5, train c1/c2/c3/c6
fold c5: test c5, val c6, train c1/c2/c3/c4
fold c6: test c6, val c1, train c2/c3/c4/c5
```

最终报告每折结果和平均结果。

## 当前优先级

1. 先把三分类伪标签做可靠，不要急着训练大模型。
2. 先证明稳定切削段预测优于完整波形预测。
3. 再比较不同回归网络。
4. 最后再强化分割模型和论文故事。
